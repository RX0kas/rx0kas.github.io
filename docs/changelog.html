Changelog

History of changes, fixes and new features across all versions.

Note
Only the minor updates are written before this version

v0.4.0 — (14-03-2026)

Added

  • First version of batch rendering
  • New MeshManager and MeshRegistry class

Changed

  • Logs will now appear only on debug mode
  • Vulkan validation layers will be enable only on debug mode
  • Debug graphics widget will only show on debug mode

Fixed

  • Fix linux drivers freeze
  • Minor Vulkan fix

Removed

  • Remove Own and Ref

v0.3.0 — (14-03-2026)

Added

  • Profiling
  • OS Independent Window Class
  • New JSON and Cache class

Changed

  • Stop using raw pointer

Fixed

  • Fix compilation warnings

Removed

  • Remove Windows and Linux Window class

v0.2.0 — (04-03-2026)

Added

  • Texture support
  • New CameraController

Changed

  • Alpha Blending enable

Fixed

  • Fix resize error

v0.1.0 — (19-02-2026)

Added

  • Universal Uniform function
  • Runtime shader compilation
  • New ImGuiWidget class
  • New ShaderType Enum
  • New Window.getAspect(), Window.show(), Window.hide() and Window.isVisible() functions
  • New Shader.isAffectedByCamera() function

Changed

  • LayerStack take shared ptr instead of raw ptr
  • Renderer::BeginScene() now return nothing
  • The window will not show directly on creation but when the run function is called

Removed

  • RenderCommand::clear() was removed
  • Model::create(const std::string &filepath, const BufferLayout& layout) was removed
